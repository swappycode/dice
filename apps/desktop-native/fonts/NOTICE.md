# Bundled fonts

These fonts are embedded into the `dice-native` binary (`src/main.rs`,
`register_fonts`) so the client renders without relying on system fonts. All are
licensed under the **SIL Open Font License 1.1** (OFL), which permits bundling and
redistribution.

| Font | Files | Source |
|---|---|---|
| **Chakra Petch** (display) | `ChakraPetch-{Regular,SemiBold,Bold}.ttf` | https://fonts.google.com/specimen/Chakra+Petch |
| **Space Mono** (monospace) | `SpaceMono-{Regular,Bold}.ttf` | https://fonts.google.com/specimen/Space+Mono |
| **Noto Emoji** (monochrome emoji fallback) | `NotoEmoji-VF.ttf` | https://fonts.google.com/noto/specimen/Noto+Emoji |

Full OFL text: https://openfontlicense.org/open-font-license-official-text/
